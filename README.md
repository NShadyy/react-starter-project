# ReactJS Starter Project

## Description

Starter project for [ReactJS](https://reactjs.org/).

## Installation

```bash
# install dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm start
```

## Building the app

```bash
# production build
$ npm run build
```

## utility scripts

```bash
# formatting files
$ npm run format

# check lint
$ npm run lint

# fix linting errors
$ npm run lint-fix
```
